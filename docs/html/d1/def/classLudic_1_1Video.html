Detailed Description

Class responsible for managing all the resources of video SGL.

Class responsible for managing all the resources of video SGL. The class is a singleton, ie, allows only a single instance of itself. Through it, you have access to all relevant routines (screen refresh, positioning, and other routine events) for managing video SGL.
