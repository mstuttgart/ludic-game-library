Detailed Description

Class to create native Allegro configuration file.

This class allow create a simple text file in the format [section] key = value

Exemple: [Monster] life = 100 power = 20 agility = 30 magic = 0

The user can write and read values how to need. It is also possible to merge two different configuration files.
